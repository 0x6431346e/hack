The Aragon client

The Aragon client is a full dapp that can create and manage decentralized organizations by running Aragon apps inside it.

The client provides Aragon app developers with:

  • Sandboxing: The client is running code from third party developers, so in order to mitigate risk (such as cross-site scripting, phishing attempts by manipulating the DOM) we sandbox apps
  • App listing: It traverses the organization to find all the relevant apps for it
  • Transaction pathing: It checks with the ACL if the user can perform an action (e.g. withdrawing funds) and if not, it tells the user possible paths to perform it (e.g. maybe by opening a voting)
  • Human readable transactions: It uses Radspec and describes the user their action in a human readable way
  • Notifications: Aragon apps can send notifications to the user when something relevant happens
